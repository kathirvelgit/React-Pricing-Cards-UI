# 💳 React Pricing Cards UI

A modern and responsive **Pricing Cards UI** built with **React.js**. This project demonstrates reusable React components, props, conditional rendering, dynamic data mapping, and responsive CSS styling. It displays three pricing plans (**Free**, **Plus**, and **Pro**) with feature availability using Font Awesome icons.

---

## 🚀 Features

- ⚛️ Built with React Functional Components
- 🧩 Reusable Pricing Card Component
- 📦 Dynamic Pricing Data using JavaScript Objects
- 🔄 Render Cards using `map()`
- 📋 Render Features Dynamically
- ✅ Conditional Rendering for Available/Unavailable Features
- 🎨 Modern Gradient Background
- 📱 Fully Responsive Design
- 🔤 Google Fonts Integration
- ⭐ Font Awesome Icons
- 🎯 Clean and Maintainable Code Structure
